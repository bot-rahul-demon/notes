UNIT 6: DATA LINK LAYER  [HIGH WEIGHTAGE]
--------------------------------------------------------------------------------
6.1 DLL Functions
    - Framing (character count, byte/character stuffing, bit stuffing)
    - Services to network layer
6.2 Error Detection & Correction
    - Types of errors (single-bit, burst)
    - Parity check (simple, two-dimensional)
    - Checksum
    - Cyclic Redundancy Check (CRC)
    - Hamming Code (error correction)
6.3 Flow Control & Error Control (ARQ Protocols)
    - Stop-and-Wait protocol
    - Stop-and-Wait ARQ
    - Sliding Window protocol
    - Go-Back-N ARQ
    - Selective Repeat ARQ
    - Piggybacking
    - Efficiency/utilization calculations
6.4 Medium Access Control (MAC) Sublayer
    - Channel allocation (static & dynamic)
    - Random Access:
        * Pure ALOHA & Slotted ALOHA (efficiency)
        * CSMA, CSMA/CD (used in Ethernet)
        * CSMA/CA (used in Wi-Fi)
    - Controlled Access: Reservation, Polling, Token Passing
    - Channelization: FDMA, TDMA, CDMA
6.5 IEEE Standards & Protocols  [from syllabus]
    - IEEE 802.3 (Ethernet - CSMA/CD)  [from syllabus]
    - IEEE 802.4 (Token Bus)  [from syllabus]
    - IEEE 802.5 (Token Ring)  [from syllabus]
    - IEEE 802.11 (Wireless LAN / Wi-Fi)
    - Standard, Fast, Gigabit Ethernet
6.6 Wired & Wireless LAN
    - Ethernet frame format
    - MAC addressing
    - Bluetooth, Wi-Fi basics

--------------------------------------------------------------------------------
7.1 Network Layer Functions
    - Logical addressing, routing, packetizing
    - Services (connection-oriented vs connectionless)
7.2 IPv4 Addressing  [from syllabus]
    - IP address structure (32-bit, dotted decimal)
    - Classful addressing (Class A, B, C, D, E)  [from syllabus]
    - Network ID and Host ID
    - Special addresses (loopback, broadcast, private, APIPA)

    - Subnet Mask
    - Subnetting (FLSM and VLSM)
    - CIDR (Classless Inter-Domain Routing)
    - Supernetting
7.4 IPv4 Datagram & Protocols
    - IPv4 header format & fields

    - ARP, RARP, ICMP, IGMP
    - NAT (Network Address Translation)

7.5 IPv6
    - Need for IPv6, 128-bit addressing
    - IPv6 header, address types
    - IPv4 vs IPv6  [from syllabus mentions IP basics]
    - Transition mechanisms (dual stack, tunneling, translation)
7.6 Routing
    - Routing vs Forwarding
    - Static vs Dynamic routing
