Operating System
================
Coal India Limited (CIL) - Management Trainee | Systems Discipline (Paper-II, Professional Knowledge)
EXHAUSTIVE TOPIC & SUBTOPIC BREAKDOWN | HIGH-EMPHASIS SUBJECT

Exam note: CIL MT is an objective (MCQ) computer-based test, conceptual-to-moderate
difficulty (easier than GATE). Emphasis on definitions, classifications, algorithm
steps, and standard numericals (scheduling, paging, disk). Items tagged
"(advanced - awareness)" are low priority. Cover every leaf point; none left.

PART A: OS FUNDAMENTALS
==================================================================

1. Introduction
   1.1 Definition, goals, and functions of an OS
   1.2 OS as resource manager and extended machine
   1.3 Types: batch, multiprogramming, time-sharing, real-time, distributed (concept)
   1.4 Multiprogramming vs multitasking vs multiprocessing
   1.5 OS structures: monolithic, layered, microkernel (concept)
   1.6 Kernel mode vs user mode; privileged instructions; dual-mode

2. System Calls and Services
   2.1 System call concept and mechanism (trap to kernel)
   2.2 User-to-kernel mode transition
   2.3 Categories: process control, file, device, information, communication
   2.4 Examples: fork, exec, wait, exit, open, read, write, close
   2.5 API vs system call (concept)
